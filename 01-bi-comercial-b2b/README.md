# Case 01 — BI Comercial (B2B consultivo)

Projeto de portfólio com **dados simulados** (sem dados reais), focado em **RevOps / BI Comercial**.

## Objetivo
Dar previsibilidade e clareza para a operação comercial com uma visão executiva do funil e dos resultados.

## KPIs
- Pipeline por etapa
- Conversão por etapa
- Receita (Won) por mês
- Ticket médio (Won)
- Forecast (previsão ponderada)
- Aging (tempo parado na etapa)

## Stack
Power BI • Excel • SQL

## Entregáveis
- CSVs simulados (pasta `data/`)
- Prints do dashboard (pasta `dashboard/`)
